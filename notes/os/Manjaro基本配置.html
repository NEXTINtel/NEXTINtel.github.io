Manjaro OS基本配置

        作为Arch Linux的衍生版本，Manjaro继承了Linux的安全稳定，同时摒弃了Arch对用户的非友好，提供了优质的GUI。Manjaro一共有4个版本：XFCE、GNOME、KDE、Architect，让用户可以有不同的选择，提供了超高的自由度。下面介绍安装KDE版本后的一些基本配置：

Manjaro安装时的一些注意事项

1：含N卡的计算机安装时卡logo无法进入

        解决方法：(1)开机进入系统引导界面后，按“e”进入grub编辑模式，
                            (2)找到quiet,在其后空一格添加"nouveau.modeset=0"，
                            (3)按F10或Ctrl+x退出编辑

2：安装好系统避免每次进入系统都操作上一步

        解决办法：(1)打开etc/default/grub
                            (2)在GRUB_CMDLINE_LINUX_DEFAULT的属性中空一格添加“pci=noacpi”
                            (3)终端执行：sudo grub-mkconfig

3：Manjaro安装/更新完成后重启报错：can't access tty,job control turned off

        Manjaro解决tty报错

Manjro安装后的配置步骤

1：切换软件源，提高网速，防止后面更新系统时出现超时bug以及UI美化时主题难以下载

        (1)执行：sudo pacman-mirrors -i -c China -m rank

        (2)弹出选择框后，全部勾选即可

        (3)打开/etc/pacman.conf文件，粘贴下面3行代码，然后保存:
              [archlinuxcn]
              SigLevel = Optional TrustedOnly
              Server=https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch

        (4)执行：sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring

2：通过开始菜单中的"添加/删除软件"来更新系统，其它方式目前会出现bug

3：更新系统后会出现中文显示乱码，用以下命令修复：

        pacman -S wqy-microhei

4：重启系统（此步必须完成，否则后面会打不开文件夹）

5：安装搜狗中文输入法

        (1)执行：sudo pacman -S qtwebkit-bin

        (2)执行：sudo pacman -S fcitx-sogoupinyin fcitx-configtool fcitx-im fcitx-sogoupinyin

        (3)执行：sudo pacman -U https://arch-archive.tuna.tsinghua.edu.cn/2019/04-29/community/os/x86_64/fcitx-qt4-4.2.9.6-1-x86_64.pkg.tar.xz

        (4)打开/etc/environment，粘贴如下内容,并保存：
              export GTK_IM_MODULE=fcitx
              export QT_IM_MODULE=fcitx
              export XMODIFIERS="@im=fcitx"

        (5)执行：qtconfig-qt4

        (6)在出现的图形化界面中找到Interface选项，找到Defult Input Method并将其设置为fcitx，点击左上角保存即可

        (7)重启计算机即可正常使用

6：打开“系统设置”，进行系统UI美化

7：用yaourt命令代替pacman命令(让命令简洁化)：

        sudo pacman -S yaourt

8：配置java环境

        (1)终端输入命令：yaourt jdk

        (2)选择想要安装的jdk版本即可

        (3)最后输入java以验证java环境是否配置成功

9：安装chrome

        yaourt -S google-chrome

10：安装vim

        yaourt -S vim

11：安装visual studio code

        yaourt -S visual-studio-code-bin

12：安装网易云音乐

        yaourt -S netease-cloud-music

13：安装teamviewer

        yaourt -S teamviewer

        备注：如果无法连接到计算机请执行如下步骤(执行前请彻底关闭teamviewer)：
              (1)执行：sudo systemctl stop teamviewerd.service
              (2)执行：sudo systemctl start teamviewerd.service

14：安装VMware Pro

        (1)执行：sudo pacman -S fuse2 gtkmm pcsclite libcanberra

        (2)查看内核版本：uname -r

        (3)执行(注意选择第(2)步中指定内核版本)：sudo pacman -S linux-headers

        (4)执行(如果已有，可不执行)：yaourt -S yay

        (5)执行：yay -S --noconfirm --needed ncurses5-compat-libs

        (6)执行：yay -S vmware-workstation

        (7)为相应的服务设置开机启动：sudo systemctl enable vmware-networks.service vmware-usbarbitrator.service vmware-hostd.service

        (8)开启相应服务：sudo systemctl start vmware-networks.service vmware-usbarbitrator.service vmware-hostd.service

        (9)查看服务状态：sudo systemctl status vmware-networks.service vmware-usbarbitrator.service vmware-hostd.service

        (10)重启系统以加载VNware模块

15：安装Jetbrains系列软件(以idea为例)

        (1)官网下载idea安装包(后缀名：xxx.tar.gz)idea安装包下载

        (2)在opt下创建Intellij文件夹：sudo mkdir /opt/Intellij

        (3)解压到创建好的的指定文件夹：sudo tar -zxvf xxx.tar.gz -C /opt/Intellij

        (4)切换到解压后的文件夹：cd /opt/Intellij/xxx

        (5)给脚本安装程序赋予执行权限：sudo chmod a=+rx bin/idea.sh

        (6)执行脚本安装程序：bin/idea.sh

        注意：在GUI安装界面注意勾选勾选Create a desktop entry for integration with system application menu 和 for all users (requires superuser privileges) 以使得可以通过图标来使用idea

16：安装kvm:

        (1)安装KVM所需的所有软件包：sudo pacman -S qemu virt-manager virt-viewer dnsmasq vde2 bridge-utils openbsd-netcat

        (2)安装ebtables和iptables软件包：sudo pacman -S ebtables iptables

        (3)打开etc/pacman.conf,粘贴如下命令并保存：
              [archlinuxfr]
              SigLevel = Never
              Server = http://repo.archlinux.fr/$arch

        (4)更新Pacman数据库：sudo pacman -Syy

        (5)使用AUR帮助程序安装libguestfs工具：yaourt -S --noconfirm --needed libguestfs

        (6)使libvirtd服务开机启动：sudo systemctl enable libvirtd.service

        (7)启用libvirtd服务：sudo systemctl start libvirtd.service

        (8)打开/etc/libvirt/libvirtd.conf，替换或增添如下内容：
              将UNIX域套接字组所有权设置为libvirt：unix_sock_group = "libvirt"
              设置R/W套接字的UNIX套接字权限：unix_sock_rw_perms = "0770"

        (9)将当前用户帐户添加到libvirt组：
              sudo usermod -a -G libvirt $(whoami)
              newgrp libvirt

        (10)重启libvirt守护进程：sudo systemctl restart libvirtd.service

        (11)启用虚拟化嵌套：
              sudo modprobe -r kvm_intel
              sudo modprobe kvm_intel nested=1
              echo "options kvm-intel nested=1" | sudo tee /etc/modprobe.d/kvm-intel.conf 选择Y即可

17：安装Android Studio

        yaourt -S android-studio

18：安装sublime text

        (1)安装GPG key：curl -O https://download.sublimetext.com/sublimehq-pub.gpg && sudo pacman-key --add sublimehq-pub.gpg && sudo pacman-key --lsign-key 8A8F901A && rm sublimehq-pub.gpg

        (2)选择稳定版本：echo -e "\n[sublime-text]\nServer = https://download.sublimetext.com/arch/stable/x86_64" | sudo tee -a /etc/pacman.conf

        (3)更新并安装：sudo pacman -Syu sublime-text
